# Chrome Setup for Thundarr User

This directory contains scripts and configuration for setting up Chrome browser for the thundarr user.

## Files

- `install-chrome.sh`: Script to install and configure Chrome
- `uninstall-chrome.sh`: Script to remove Chrome and its configuration

## Installation

To install Chrome for the thundarr user:

```bash
./install-chrome.sh
```

## Uninstallation

To remove Chrome and its configuration:

```bash
./uninstall-chrome.sh
```
